# Exit 8 - Inspired Sketch

## Project Title: Exit 8 - Inspired Sketch

## Setup and Interaction Instructions:
Open up Github "pages" link in Google Chrome. 
The sketch displays an exit 8 inpired enviornment 
with tiled walls, yellow exit 8 poster with black text and the classic eye poster.
I used p5.js from both Open Processing and Visual Studio Code to create this sketch.
